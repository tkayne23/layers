# Required
- Node.js > v6.10
- npm

# Tools
- Angular CLI
  ```
  npm install -g @angular/cli
  ```

# app
### setup
1. `cd app`
2. `npm install`

### server local
1. `grunt`

### test
1. `grunt test`

# cloud
### setup
1. `cd cloud`
2. `npm install`
3. (temp) `cd services/etl && npm install`

### playing with things (because I didn't finish the config)
